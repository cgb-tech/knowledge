# git分支相关命令

## 1、创建分支

- git branch <分支名>
- git branch -v 查看分支名

## 2、切换分支

- git checkout <分支名>
- 一步完成：git checkout -b <分支名>

## 3、合并分支

- 先切换到主干 git checkout master
- git merge <分支名>

## 4、删除分支

- 先切换回主干 git checkout master
- git branch -D <分支名>
