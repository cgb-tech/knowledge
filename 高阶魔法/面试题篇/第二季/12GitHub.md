# github

## 1、常用词

1. watch
2. fork
3. star
4. clone
5. follow

## 2、in关键词限制搜索范围

```shell
seckill in:name
seckill in:description
seckill in:readme
seckill in:read,readme
```

## 3、stars或fork数量关键字去查找

```shell
xxx stars :>|:>=
springboot stars:>=5000
xxx stars 1..2
```

```shell
springboot forks:>=5000
xxx forks 1..2
```

## 4、awesome加强搜索

```shell
awesome 关键字
```

- 一般是用来收集学习、工具、书籍类相关的项目

## 5、给别人指出关键代码的行号
