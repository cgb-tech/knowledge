# GC篇

## 常见的垃圾回收算法

### 1. 引用计数

- 有引用的时候+1 ，少引用的时候-1 ，到0回收
- 无法处理循环引用的问题
- 现在已经不再使用

### 2. 复制算法

- 复制->清空->互换
- 不会产生碎片
- 浪费空间，大对象复制起来耗时

### 3. 标记清除

- 没有大面积复制，节约空间
- 产生了碎片

- 先标记出要回收的对象，然后统一回收这些对象

### 4. 标记整理

- 标记->压缩 (Mark)->(Compact)
